# Portfolio Starter (Vite + React + Tailwind + Framer Motion)

## Quickstart
1. Open a terminal in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start dev server:
   ```bash
   npm run dev
   ```

Edit text in:
- `src/components/Hero.jsx` (your name & tagline)
- `src/components/Contact.jsx` (your email)
